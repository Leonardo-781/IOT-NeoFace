Kit minimo

Pecas necessarias

  • 1x ESP32 DevKit.
  • 1x sensor BMP280 (temperatura/pressao).
  • 1x sensor MQ-7 (gas/CO) opcional.
  • 1x LDR + resistor (luminosidade) opcional.
  • Jumpers, protoboard e cabo USB.

Firmware de referencia: pasta esp32 no repositorio oficial.

Guia visual

Diagrama basico de ligacao

ESP32 (3V3)  --------  VCC BMP280
ESP32 (GND)  --------  GND BMP280
ESP32 (GPIO21) ------  SDA BMP280
ESP32 (GPIO22) ------  SCL BMP280

ESP32 (GPIO34) ------  LDR (via divisor) [opcional]
ESP32 (GPIO35) ------  MQ-7 analogico     [opcional]

Ajuste os pinos no firmware conforme seu hardware real.
