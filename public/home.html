Plataforma

Monitoramento IoT com gestão de contas, telemetria e comandos em tempo real

O NeoFace integra ESP32, broker MQTT, API e painel web para acompanhar sensores, enviar comandos e organizar a operacao de forma simples.

Como funciona

Fluxo do sistema

  1. ESP32 coleta sensores (temperatura, umidade, luz, gas).
  2. Gateway/broker recebe e repassa dados em topicos MQTT.
  3. API processa ingestao, alertas e persistencia.
  4. Painel mostra estado da frota e permite comandos.
  5. Gestor define permissoes dos usuarios cadastrados.
