# Guia Rápido: Gerar PDF no Overleaf em 3 Minutos

## ⚡ Passo a Passo Rápido (Método Simples)

### 1️⃣ Abrir Overleaf
- URL: https://www.overleaf.com/
- Clique em **"Sign In"** ou **"Register"** (crie conta se necessário)
- Clique em **"New Project"**

### 2️⃣ Criar Projeto
- Escolha **"Blank Project"**
- Nome: `Etapa1-32021BSI004`
- Clique **"Create"**

### 3️⃣ Copiar Arquivo LaTeX
**NO SEU COMPUTADOR:**
- Abra: `etapa-1/documentos/Etapa-1-LaTeX.tex`
- Selecione tudo: `Ctrl+A`
- Copie: `Ctrl+C`

**NO OVERLEAF:**
- Clique no arquivo `main.tex` (no painel esquerdo)
- Selecione tudo: `Ctrl+A`
- Delete: `Delete`
- Cole: `Ctrl+V` (o conteúdo do arquivo LaTeX)

### 4️⃣ Compilar
- Clique no botão verde **"Recompile"**
- Aguarde a compilação (10-20 segundos)
- PDF aparece no painel direito 🎉

### 5️⃣ Baixar PDF
- Clique em **"Download"** (botão abaixo do PDF)
- Escolha **"Download PDF"**
- Salve como: `32021BSI004_Etapa1.pdf`

## 💡 Dicas Extras

✨ **Esconder Avisos Menores:** O Overleaf às vezes mostra warnings (amarelos). Pode ignorar se o PDF compila corretamente.

🔄 **Auto-Compile:** Ative em Settings (ícone engrenagem) → **"Auto Compile"** para compilar automaticamente enquanto edita.

📖 **Ver Código Fonte:** Clique em **"Editor"** para ver/editar o código LaTeX.

## 🚨 Se Houver Erro de Compilação

**Erro típico:**
```
! LaTeX Error: File not found: ...
```

**Solução:**
1. Clique no ícone de engrenagem ⚙️
2. Verifique **"TeX Live Version"** (use a mais recente)
3. Clique em **"Recompile"** novamente

**Se ainda não funcionar:**
1. Clique em **"Logs and output files"**
2. Veja a mensagem de erro específica
3. Tente recompilar (geralmente resolve na segunda tentativa)
